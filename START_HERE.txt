STEP 1: GET YOUR ANTHROPIC API KEY (2 minutes)
────────────────────────────────────────────────────────────────────────────────

1. Go to: https://console.anthropic.com/account/keys
2. Sign in or create a free account
3. Click "Create Key"
4. Copy the key (starts with "sk-ant-")
5. Paste it somewhere safe — you'll need it in Step 3

💡 TIP: You get a free trial with API credits. Perfect for testing!

════════════════════════════════════════════════════════════════════════════════

STEP 2: PUSH CODE TO GITHUB (1 minute)
────────────────────────────────────────────────────────────────────────────────

A. Create repository at: https://github.com/new
   - Repository name: starpush
   - Public (not private)
   - Leave everything else default
   - Click "Create repository"

B. Copy the HTTPS URL shown (looks like):
   https://github.com/YOUR_USERNAME/starpush.git

C. Run this in Terminal:

   cd "/Users/skelemer/Desktop/Files/Projects & Code/b2b seo"
   git remote add origin https://github.com/YOUR_USERNAME/starpush.git
   git branch -M main
   git push -u origin main

   (Replace YOUR_USERNAME with your actual GitHub username)

✓ Code is now on GitHub!

════════════════════════════════════════════════════════════════════════════════

STEP 3: DEPLOY TO RENDER (3 minutes)
────────────────────────────────────────────────────────────────────────────────

1. Go to: https://dashboard.render.com (sign in with GitHub)
2. Click "New +" button (top right)
3. Click "Web Service"
4. Click "Deploy existing repository"
5. Select your "starpush" repository
6. Fill in the form:
   - Name: starpush
   - Environment: Node
   - Build Command: npm install
   - Start Command: node server.js
7. Click "Advanced" (at bottom)
8. Add Environment Variable:
   - Key: ANTHROPIC_API_KEY
   - Value: [paste your API key from Step 1]
9. Click "Create Web Service"
10. Wait 3-5 minutes for deployment
11. ✅ YOUR LIVE URL APPEARS IN THE DASHBOARD!

IF SOMETHING GOES WRONG

Most common issues:

❌ "Deployment failed"
   → Check your ANTHROPIC_API_KEY is set correctly in Render
   → Check the build logs in Render dashboard

❌ "Can't access the URL"
   → Wait 2-3 more minutes for deployment to fully complete
   → Refresh the page
   → Check Render dashboard for status

❌ "GBP Optimizer isn't working"
   → Make sure ANTHROPIC_API_KEY is set in Render
   → API must be valid and have available credits
   → Check Render logs for errors
