STEP 3: Deploy to Render (2 minutes)
───────────────────────────────────

Go to: https://dashboard.render.com
1. Click "New +" button
2. Select "Web Service"
3. Select "Deploy existing repository"
4. Choose your "reviewpilot" repo
5. Fill in:
   - Name: reviewpilot
   - Environment: Node
   - Build Command: npm install
   - Start Command: node server.js
6. Click "Advanced" and add environment variable:

   - Value: [paste your API key from Step 1]
7. Click "Deploy Web Service"
8. Wait 3-5 minutes
9. YOUR LIVE URL appears in the Render dashboard!
